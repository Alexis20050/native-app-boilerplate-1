# native-app-boilerplate

## Running the app

### creating virtual environment
```
python3 -m venv venv
```
or
```
python -m venv venv
```
### Activate virtual environment
For windows
```
.\venv\Scripts\activate
```
For mac or linux
```
source venv/bin/activate
```

### Install Dependencies
```
pip install -r requirements.txt
```

### Running the App
```
python -m app.main
```
